Frequently Asked Questions

What if I don't have the LocQar app?▼
No worries! You'll get an SMS with a unique code. No app needed to receive.
How long can my package stay?▼
48–72 hours free storage. After that, a small daily fee applies.
Is there a weight or size limit?▼
No weight limits! If it fits in the locker, it ships.
Can businesses use LocQar?▼
Absolutely! We offer dedicated business plans. Visit our Business Portal →
How do I pay?▼
Mobile money (MTN MoMo, Vodafone Cash, AirtelTigo Money), debit/credit cards, and in-app wallet.

Developer-Friendly

Integrate in Minutes

RESTful API, webhooks, and pre-built plugins for popular platforms.

send_package.py
# Send a package via LocQar API
import locqar

client = locqar.Client(api_key="lq_live_...")

package = client.packages.create(
    sender_phone="+233241234567",
    receiver_phone="+233509876543",
    station="accra-mall",
    size="medium",
    notify=True
)

print(f"Code: {package.pickup_code}")
# Output: Code: LQ-8294
